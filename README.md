# Moral Money Ecosystem

Moral Money is a research project exploring how decentralized communities can organize governance, reputation and economic activity using blockchain infrastructure.

The system is structured in layers:

Constitution  
→ Community Model  
→ Reputation System  
→ Blockchain Implementation

## Repository Structure

ai-agents/  
Experimental AI agents for governance and evaluation.

blockchain-core/  
Substrate-based blockchain implementation.

docs/  
Institutional and system documentation.

frontend/  
Interface experiments.

## Core Idea

Authority in the system must emerge from **real contributions to the survival and prosperity of the community**.

Contributions may include:

- Work
- Capital
- Knowledge
- Infrastructure
- Services
