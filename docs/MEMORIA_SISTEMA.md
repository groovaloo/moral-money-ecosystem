# 🧠 Memória Mestra do Ecossistema Moral Money

## 1. A Visão do Fundador (Nuno Cunha)
* **Não à Utopia Inútil:** Recusa de uma comunidade de "hippies" sem produtividade.
* **Não à Plutocracia:** Recusa de uma comunidade de "ricos" onde o dinheiro compra poder de decisão.
* **Justiça de Impacto:** O valor de um membro é medido pelo benefício real para a sobrevivência coletiva.
* **Equivalência Ética:** Um investidor (ex: 60 anos, 4M€ em infraestrutura/irrigação) tem o mesmo valor e soberania que um trabalhador ativo, pois ambos viabilizam a prosperidade do grupo.

## 2. Pilares do Projeto
* **Habitação:** LSF (Light Steel Frame), modular e expansível conforme o mérito/necessidade, sem estética "comunista" (casas iguais).
* **Economia:** Buildcoin via Blockchain (Substrate) para registo de ativos e reputação.
* **Governação:** Decisões técnicas por grupos de 5 especialistas validados pelo sistema.
* **Tecnologia:** IA Local (Ollama/Llama3) atuando como auditor ético e gestor de recursos.

## 3. Estado Atual do Sistema (2026-03-07)
* **Hardware:** MacBook Pro 2018 (i7, 16GB RAM) validado como capaz.
* **Ambiente Blockchain:** Rust e Wasm instalados.
* **Ambiente IA:** Python venv configurado; Ollama funcional com Llama3 (8B) carregado.
* **Primeiro Protótipo:** Criado o `agente_etica.py` para validar a equivalência Investidor/Trabalhador.

## 4. Notas Importantes para a Próxima Sessão
* O sistema deve ser exequível e documentado para atrair investidores/crowdfunding.
* A IA deve ser o "árbitro cego" que garante que o dinheiro não compra influência política.

## Estrutura Modular e Custo (2026-03-07)
* **Industrialização:** A estrutura modular é a única forma de atingir o menor custo possível através da produção em escala.
* **Flexibilidade de Escala:** Permite fazer uma casa pequena ou uma casa grande usando os mesmos componentes base.
* **Evolução:** Quem precisa de mais espaço ou contribui mais para a comunidade pode simplesmente adicionar novos módulos à sua habitação existente.
